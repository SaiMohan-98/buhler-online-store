This project is a simple single-page online store application built as part of a frontend assignment.
The goal was to create a clean, functional UI where users can browse products, view details, add items to cart, and proceed to checkout.

**Features**

Product listing grouped by categories (Rollers, Optical Sorters, Dryers, etc.)

Product details page with route-based navigation

Add to cart functionality

Cart count visible in header

Checkout page with selected products and total price

Client-side routing

Fully working demo hosted on GitHub Pages


**Tech Stack**

Vue 3

Vite

TypeScript

**Notes**

Category labels are centralized in a shared constants file to avoid duplication.

Routing uses hash mode to support GitHub Pages hosting.

The app is fully client-side and does not rely on any backend APIs.

Vue Router

Pinia (for cart state management)

GitHub Pages (for hosting)
